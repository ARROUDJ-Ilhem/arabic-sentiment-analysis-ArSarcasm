# Arabic Dialect Sentiment Analysis — ArSarcasm

Fine-tuning of **CAMeLBERT-Mix** for 3-class sentiment classification (positive / negative / neutral) on the ArSarcasm dataset (~10,500 Arabic tweets in MSA and dialectal Arabic).

## Results

| Model | Test Macro-F1 |
|---|---|
| BiLSTM (baseline) | 0.6134 |
| CAMeLBERT-Mix (lr=5e-5) | unstable |
| **CAMeLBERT-Mix (lr=2e-5)** | **0.7396** |

Val loss: 0.5802 · Val macro-F1: 0.7447 · Val accuracy: 0.7735  
Best checkpoint: `checkpoint-690` (epoch 3, early stopping patience=2)  
Inference latency: mean **13.51 ms**, p95 **20.10 ms** (Tesla T4, single sample)

## How to run

### 1. Local (no GPU needed)

```bash
# Create virtual env
python -m venv venv
venv\Scripts\activate          # Windows
# source venv/bin/activate     # Linux/macOS

# Install local deps
pip install torch --index-url https://download.pytorch.org/whl/cpu
pip install pyarabic pandas numpy scikit-learn matplotlib seaborn pytest

# Sanity check preprocessing
python src/preprocess.py

# Download + clean + split dataset (ArSarcasm, MIT license)
python scripts/prepare_data.py

# Run unit tests
pytest tests/ -v
```

### 2. Training on Google Colab (GPU T4)

Open `notebooks/training_colab.ipynb` on [colab.research.google.com](https://colab.research.google.com).  
Set runtime to **T4 GPU** → Run all. Duration: ~3–4 min.

## Dataset

[ArSarcasm](https://github.com/iabufarha/ArSarcasm) — 10,547 Arabic tweets (MSA + Gulf, Levantine, Egyptian, Maghrebi dialects).  
Sentiment labels: positive / negative / neutral. License: **MIT**.

## Model

`CAMeL-Lab/bert-base-arabic-camelbert-mix-sentiment` — BERT-base pre-trained on MSA + dialectal Arabic (Twitter, Wikipedia, news). License: **Apache 2.0**.

Chosen over:
- **AraBERT** — pre-trained on MSA only, weaker on dialectal negation
- **BiLSTM** — no pre-trained Arabic representations, macro-F1 = 0.61

## Limitations

ArSarcasm is dominated by Gulf and Levantine dialects. Maghrebi dialects (Algerian, Moroccan Darija) represent < 2% of the corpus (183 / 10,547 tweets). Performance on Algerian dialect is likely lower than the reported macro-F1.
